Editor
UEditor的核心类，为用户提供与编辑器交互的接口。
事件
  • selectionchange
    描述
    • 每当编辑器内部选区发生改变后， 将触发该事件
    • 该事件的触发非常频繁，不建议在该事件的处理过程中做重量级的处理
     参数列表
    1. editor
      UE.Editor当前所监听的编辑器对象的引用
方法
  • ready (Function  fn)
    描述
    参数列表
    1. fn
      Function编辑器ready之后所执行的回调, 如果在注册事件之前编辑器已经ready，将会
     示例 
    1
2
3
    editor.addListener( 'ready', function( editor ) {
    editor.setContent('初始化完毕');
} );
  • setOpt (PlainObject  settings)
    描述
    • 以key-value集合的方式设置编辑器的配置项，以覆盖编辑器的默认选项值
    • 该方法仅供编辑器构造函数调用，其他任何方法不能调用。
     参数列表
    1. settings
      PlainObject编辑器的可接受的选项的key-value集合
     示例

    javascript editor.setOpt( { 'initContent': '欢迎使用编辑器' } );

  • destroy ()
    描述
    • 销毁编辑器实例对象
     示例

    javascript editor.destroy();

  • getContent (Function  fn)
    描述
    • 获取编辑器的内容。 可以通过参数定义编辑器内置的判空规则
    • 该方法获取到的是经过编辑器内置的过滤规则进行过滤后得到的内容
    • 该方法在处理包含有初始化内容的时候能起到很好的作用。
     参数列表
    1. fn
      Function自定的判空规则， 要求该方法返回一个boolean类型的值，
     返回值
    • String编辑器的内容字符串
     示例

    javascript // editor 是一个编辑器的实例 var content = editor.getContent( function ( editor ) { return editor.body.innerHTML === '欢迎使用UEditor'; } );
