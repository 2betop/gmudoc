  • Editor
Editor
UEditor的核心类，为用户提供与编辑器交互的接口。
事件
  • ready
    描述
    参数列表
    1. editor
      UE.Editor当前所监听的编辑器对象的引用
     示例 
    editor.addListener( 'ready', function( editor ) {
    editor.execCommand( 'focus' );
} );
  • selectionchange
    描述
    • 该事件的触发非常频繁，不建议在该事件的处理过程中做重量级的处理
     参数列表
    1. editor
      UE.Editor当前所监听的编辑器对象的引用
方法
  • ready(Function  fn)
    描述
    参数列表
    1. fn
      Function编辑器ready之后所执行的回调, 如果在注册事件之前编辑器已经ready，将会
     示例 
    editor.addListener( 'ready', function( editor ) {
    editor.setContent('初始化完毕');
} );
  • setOpt(String  key, *  val)
    描述
    • 该方法仅供编辑器构造函数调用，其他任何方法不能调用。
     参数列表
    1. key
      String编辑器的可接受的选项名称
    2. val
      *该选项可接受的值
     示例 
    editor.setOpt( 'initContent', '欢迎使用编辑器' );
  • setOpt(PlainObject  settings)
    描述
    • 该方法仅供编辑器构造函数调用，其他任何方法不能调用。
     参数列表
    1. settings
      PlainObject编辑器的可接受的选项的key-value集合
     示例 
    editor.setOpt( {
    'initContent': '欢迎使用编辑器'
} );
  • destroy()
    描述
    示例 
    editor.destroy();
  • getContent()
    描述
    • 该方法获取到的是经过编辑器内置的过滤规则进行过滤后得到的内容
     返回值
    • String编辑器的内容字符串, 如果编辑器的内容为空， 则返回空字符串
     示例 
    var content = editor.getContent();
  • getContent(Function  fn)
    描述
    • 该方法获取到的是经过编辑器内置的过滤规则进行过滤后得到的内容
    • 该方法在处理包含有初始化内容的时候能起到很好的作用。
     参数列表
    1. fn
      Function自定的判空规则， 要求该方法返回一个boolean类型的值，
     返回值
    • String编辑器的内容字符串
     示例 
    // editor 是一个编辑器的实例
var content = editor.getContent( function ( editor ) {
     return editor.body.innerHTML === '欢迎使用UEditor';
} );
