  • getContent (Function  fn)
    • 获取编辑器的内容。 可以通过参数定义编辑器内置的判空规则
    • 该方法获取到的是经过编辑器内置的过滤规则进行过滤后得到的内容
    • 该方法在处理包含有初始化内容的时候能起到很好的作用。
    参数
    名称 类型 描述
    fn Function 自定的判空规则， 要求该方法返回一个boolean类型的值，
    返回值
