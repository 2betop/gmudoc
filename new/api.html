ready(Function fn)

编辑器对外提供的监听ready事件的接口， 通过调用该方法，达到的效果与监听ready事件是一致的
参数列表
参数名 类型 描述

1
2
3
4
5

 } );

setOpt(String key, * val)

以attributeName - attributeValue的方式设置编辑器的配置项，以覆盖编辑器的默认选项值
  • 警告： 该方法仅供编辑器构造函数调用，其他任何方法不能调用。
参数列表
参数名 类型 描述
key String 编辑器的可接受的选项名称
val * 该选项可接受的值
示例 
1
 editor.setOpt( 'initContent', '欢迎使用编辑器' );

setOpt(PlainObject settings)

以key-value集合的方式设置编辑器的配置项，以覆盖编辑器的默认选项值
  • 警告： 该方法仅供编辑器构造函数调用，其他任何方法不能调用。
参数列表
参数名 类型 描述
settings PlainObject 编辑器的可接受的选项的key-value集合
示例 
1
2
3
 editor.setOpt( {
     'initContent': '欢迎使用编辑器'
 } );

getContent()

获取编辑器的内容
  • 警告： 该方法获取到的是经过编辑器内置的过滤规则进行过滤后得到的内容
示例 
1
 var content = editor.getContent();

getContent(Function fn)

获取编辑器的内容。 可以通过参数定义编辑器内置的判空规则
  • 提示： 该方法在处理包含有初始化内容的时候能起到很好的作用。
  • 警告： 该方法获取到的是经过编辑器内置的过滤规则进行过滤后得到的内容
参数列表
参数名 类型 描述
fn Function 自定的判空规则， 要求该方法返回一个boolean类型的值， 代表当前编辑器的内容是否空， 如果返回true， 则该方法将直接返回空字符串；如果返回false，则编辑器将返回 经过内置过滤规则处理后的内容。
示例 
1
2
3
4
5
6
 // editor 是一个编辑器的实例
 var content = editor.getContent( function ( editor ) {

      return editor.body.innerHTML === '欢迎使用UEditor';

 } );
